--------------------------------------词法与语法要求--------------------------------------
1. 没有负数

--------------------------------------键盘输入测试用例--------------------------------------

1.无浮点数、无注释时
    （1）各个符号单个输入
    if,else,while,read,write,int,real
    +,-,*,/,=,<,>,==,<=,>=,<>,{,},[,]

    （2）语句（有空白与无空白）
    int a, b;
    real[] r;
    a = 1;
    read(b);
    r[0] = 0;

    while(a < b){
        if(a == 3){
            r[3] = 3;
        }else if(a >= 5){
            r[5] = 5;
        }else{
            r[7] = (a * 3) - (b / 3);
        }
        a = a + 1;
    }

    write(a, b, r);

    （3）出错处理
    a. 不正确的标识符，如_a，1a，a1b_

（1）单行注释
    a. //read(a);

    a. [多行注释]
